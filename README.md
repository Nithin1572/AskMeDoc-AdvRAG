## Getting Started

### Prerequisites
- Python 3.10+
- [Ollama](https://ollama.com) installed on your system

### Steps
1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies from `requirements.txt`
4. Pull the local model via Ollama
5. Run the project
